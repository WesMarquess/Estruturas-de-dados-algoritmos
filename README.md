📚 Estruturas de Dados e Algoritmos (DSA)

Repositório de estudos baseado no curso de Estruturas de Dados e Algoritmos da Loiane Groner disponível no YouTube.

Este projeto tem como objetivo consolidar conhecimentos fundamentais em DSA por meio de implementações práticas, exercícios e anotações pessoais.

🎯 Objetivo

Compreender os principais conceitos de Estruturas de Dados

Implementar algoritmos do zero

Praticar análise de complexidade (Big-O)

Resolver exercícios práticos

Criar uma base sólida para entrevistas técnicas
